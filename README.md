# 🚀 Pxoqxo.UnitTest

**Pxoqxo.UnitTest** is a lightweight ⚡ unit testing framework for .NET, written entirely in 💙 C#. It is built for developers who want fast, simple, and dependency-free unit testing without unnecessary complexity.

The framework supports both 🔄 synchronous and ⚡ asynchronous test execution. Test methods return a ✅ `bool` value to indicate success or failure, while the framework provides detailed execution information, including 📚 stack traces and clear failure types such as ❌ condition failures or 💥 unexpected exceptions.

## ✨ Features

- ⚡ Lightweight with zero external dependencies.
- 💙 Built entirely in C# for .NET.
- 🚀 Fast test discovery and execution.
- 🔄 Supports both synchronous and asynchronous test methods.
- ✅ Simple `bool`-based test results.
- 📚 Detailed execution stack and diagnostic information.
- ❌ Differentiates between failed test conditions and thrown exceptions.
- 🛠️ Clean and minimal API with low overhead.
- 🔓 Fully Free and Open Source (FOSS).

## 🎯 Goals

Pxoqxo.UnitTest aims to provide a fast, lightweight, and easy-to-use testing framework that focuses on performance, simplicity, and clear diagnostics while remaining easy to integrate into any .NET project.

+ .NET 8 SDK
